표: 가로방향-행 row, 세로방향-열 column

속성 설명
table 표를 나타내는 태그
tr 행을 나타내는 태그
th 제목 셀을 나타내는 태그
td 셀을 나타내는 태그
  • 하나 이상의 tr + tr의 자식 th, td

표의 구조 2

표의 구조와 관련된 태그
이름 설명
caption 표의 제목을 나타내는 태그
thead 제목행을 그룹화하는 태그
tbody 본문행을 그룹화하는 태그
tfoot 바닥행을 그룹화하는 태그

html 버전에 따른 tfoot 위치

  • html4: tfoot > tbody
  • html5: tfoot의 위치가 tbody 앞뒤 상관 없음
  • html5.1~: tbody > tfoot

10: 폼 요소1

폼 요소: 서버에 데이터를 전달하기 위한 요소
  • 대표적 폼 요소
  • 내용이 없는 빈 요소
  • type 속성을 통해 여러 종류의 입력양식으로 나타남

type: text

아이디, 이름, 주소, 전화번호 등 단순한 텍스트 입력할 때 사용
  • type:text의 "placeholder" 속성
  • 사용자가 입력하기 전 미리 화면에 노출하는 값
  • 입력하는 값의 양식을 표현할 수 있음

type: password

암호와 같이 공개할 수 없는 내용을 입력할 때 사용
  • 화면에는 type:text처럼 나타남
  • 실제로 입력할 때는 값을 노출하지 않음

type: radio

남자 여자
라디오 버튼을 만들 때 사용
  • 중복선택 불가
  • 하나의 항목만을 선택

type: checkbox

등산 독서 운동
체크박스 만들 때 사용
  • 중복선택 가능

추가설명

name explanation
"checked" property 값이 별도로 존재하지 않는 boolean true: 속성 O || false: 속성 X
"name" property 라디오 버튼과 체크박스를 그룹화시켜주는 속성

12: 폼 요소 3

"select" tag

선택목록상자, 콤보박스라고도 함
  • 선택지를 리스트 형태로 노출
  • 그 중 하나를 선택
  • "multiple" prop로 다중 선택
  • "option" tag로 각 항목 나타냄
  • "seleted" prop: 선택된 항목

"textarea" tag

"rows", "cols" props: 텍스트 상자 크기 조절
  • cols: 가로 크기를 조절(정확한 글자수X)
  • rows: 세로 크기 조절(화면에 보여지는 줄 수)

"button" tag

submit, reset, button: 3가지 타입
  • "input" type의 submit, reset, button과 동 기능
  • 빈 태그 X
  • 내용을 직접 넣으므로 자유로운 스타일 표현

13: 폼 요소 4

"label" tag

:
:

form 요소의 이름과 form 요소를 명시적으로 연결
  • 모든 form 요소에 사용 가능
  • form 요소의 id 속성값과 "label"의 for 속성 값을 같게
  • label을 사용하면 클릭 시 해당 form 클릭한 것처럼 동작
  • 스크린 리더기를 통해 들으면 label을 함께 읽음
  • 사용성, 접근성 측면으로 중요한 역할을 하므로 반드시 사용

"form", "fieldset", "legend" tag
